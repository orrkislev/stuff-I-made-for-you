Noodle Noodle Noodle Introduction What is this?

This is an interactive creative coding sketch that generates colorful, wavy noodle-like structures. As you interact with the sketch, it creates dynamic, flowing noodles with varying colors and shapes.

Key Features
  • Interactive noodle creation
  • Dynamic color transitions
  • Smooth, flowing curves
  • Adjustable light effects
  • Customizable noodle properties
 How It Works
  • 1. Noodles are generated with smooth curves
  • 2. Colors transition smoothly along the noodle
  • 3. Light effects are applied to enhance visual appeal
  • 4. Noodle properties such as radius and color can be customized
  • 5. The sketch uses p5.js and paper.js for rendering

The result is a visually engaging, dynamic noodle drawing that responds to user interaction and showcases smooth color transitions and light effects.
