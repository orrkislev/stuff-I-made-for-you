Interactive 3D-like Cylindrical Structures Introduction What is this?

This is an interactive creative coding sketch that generates 3D-like visualizations of cylindrical structures. As you drag your mouse, it creates colorful, organic-looking cylinders that rotate and undulate, giving the impression of woven or braided tubes.

Key Features
  • Mouse-driven creation of cylindrical bodies
  • Organic, gap-filled ring structures
  • Rotating and undulating animations
  • Simple 3D-to-2D projection for depth effect
  • Color cycling through a predefined palette
 How It Works
  • 1. Mouse input creates "bodies" composed of circular rings
  • 2. Rings are made of nodes arranged in a circular pattern
  • 3. Perlin noise influences node positions, creating organic shapes and gaps
  • 4. The structure rotates and animates with subtle undulation
  • 5. Simple 3D-to-2D projection is used for rendering
  • 6. Lines are drawn between nodes to form the cylindrical shape
  • 7. Sections are sorted by depth for a basic 3D effect
  • 8. Darker colors are applied to parts that appear to be "behind" others

The result is a visually engaging, semi-3D effect that produces organic-looking cylindrical structures through user interaction.
