Layered, natural-looking patterns Key Parameters
  1. slices: Determines the number of sections in the artwork.
  2. noiseScale: Affects the granularity of the Perlin noise used for point filtering.
  3. thresh: Influences the overall density of the patterns.

The combination of Poisson Disc Sampling, Perlin noise, and randomized drawing functions results in a complex, organic-looking texture that evolves over time. The use of sections with different properties creates distinct areas within the artwork, adding depth and visual interest.

This sketch demonstrates an approach to creative coding that combines various algorithms and techniques to produce a natural, flowing aesthetic.

 Explanation How does it work?

This sketch produces organic, textured patterns. Here's how it works:

1. Setup

  • The sketch uses a combination of p5.js and Paper.js libraries.
  • It initializes a canvas and sets up the drawing environment.

2. Point Generation

  • Uses Poisson Disc Sampling to generate a set of evenly distributed points across the canvas.
  • These points serve as the foundation for the patterns.

3. Section Creation

  • The canvas is divided into multiple sections (defined by the 'slices' parameter).
  • Each section has its own properties, including a threshold range and a drawing function.

4. Drawing Process

  • For each frame, the sketch iterates through the sections.
  • Points within each section are filtered based on Perlin noise and the section's threshold.
  • The remaining points are then drawn using the section's specific drawing function.

5. Drawing Functions

  • There are two main drawing functions:
    • point_random: Creates small clusters of random points.
    • line_noise: Draws lines with angles determined by Perlin noise.

6. Noise Layer

  • A separate graphics layer adds a subtle noise texture to the entire image.
